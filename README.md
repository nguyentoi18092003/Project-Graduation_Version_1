 # Graduation-Project
 This is a graduation project on the topic of automated testing conducted by a PTIT student.
 # OrangeHRM5 - Automated Testing Project

## Thông tin về dự án
### 1. Hệ thống Web: OrangeHRM5
- **Mã nguồn mở**: Là phần mềm giúp các tổ chức hoặc công ty quản lý và sử dụng nguồn nhân lực một cách hiệu quả.
- **Ứng dụng**: Một trong những giải pháp quản lý nhân sự hàng đầu thế giới, phù hợp với doanh nghiệp vừa và nhỏ.
- **Trạng thái**: Đã tải xuống và cài đặt trên máy cục bộ (local).
- **Cơ sở dữ liệu**: Gồm 179 bảng.
- **Các chức năng chính**: [Link Canva](https://www.canva.com/design/DAGgY9vMB1c/gUxEufqL_dFndNmMMX_cXw/edit)

### 2. Hướng tiếp cận và công nghệ áp dụng
#### 2.1. Công cụ và kỹ thuật kiểm thử tự động
- **Công cụ kiểm thử tự động**: Selenium
- **Framework kiểm thử**: TestNG
- **Mẫu thiết kế (Design Patterns) áp dụng**:
  - Page Object Pattern
  - Factory Pattern
  - Singleton Pattern
- **Ngôn ngữ lập trình**: Java
- **Công cụ xây dựng & tích hợp liên tục (CI/CD)**: Maven
- **Quản lý mã nguồn**: Git, GitHub

#### 2.2. Quản lý cơ sở dữ liệu & kiểm thử dữ liệu
- **Phương pháp kiểm thử dữ liệu (Data Driven Testing)**:
  - Sử dụng JSON, Properties File, Excel file
  - POJO (Plain Old Java Object)
- **Kiểm thử cơ sở dữ liệu**: Tích hợp Selenium với JDBC, MySQL

#### 2.3. Báo cáo & quản lý lỗi
- **Công cụ báo cáo**: Extend Report

#### 2.4. Áp dụng nguyên tắc lập trình
- **Nguyên tắc lập trình**:
  - DRY (Don't Repeat Yourself)
  - KISS (Keep It Simple, Stupid)
  - YAGNI (You Ain't Gonna Need It)
  - SOLID (Nguyên tắc thiết kế hướng đối tượng SOLID)
- **Lập trình hướng đối tượng (OOP - Object-Oriented Programming)**

#### 2.5. Kiểm thử song song và môi trường chạy thử
- **Parallel Testing & ThreadLocal**
- **Selenium GRID kết hợp với**
  - Máy thực (Real Machine)
#### 2.6. Tích hợp & Kiểm thử hiệu năng
- **Tích hợp hệ thống CI/CD**: Jenkins *(đang nghiên cứu)*
- **Kiểm thử hiệu năng với JMeter**:
  - Load Testing 
  - Stress Testing 
#### 2.7. Hướng dẫn cài đặt web
- **Cài đặt XAMPP để tạo môi trường máy chủ web (Apache, MySQL, PHP)**
- **Cài đặt OrangeHRM 5**
#### 2.8. Hướng dẫn cài đặt công cụ
- **Cài đặt Java JDK 17**
- **Cài đặt Apache Maven 3.9.9**
#### 2.9.  Hướng dẫn chạy dự án
- **src/java/resources/ chuột phải /run**
